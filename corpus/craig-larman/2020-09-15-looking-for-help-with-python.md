great that you're digging into this. if understand correctly, u r trying to
set up on win os machine. afaik and observe the vast majority of devs
(outside of the .Net world) for ML work (and most else dev these days)
avoid win OS, and instead use mac os (a berkeley unix distro) or linux. mac
os is a real unix os, and has full support for traditional unix command
line and terminal control, via the builtin Terminal app, which by default
sets up a bash shell.

suggest u avoid windows, and develop this on mac os. making that switch
will probably align your experiments with the tutorials much better. of
course, don't use any tutorial that is based on windows
